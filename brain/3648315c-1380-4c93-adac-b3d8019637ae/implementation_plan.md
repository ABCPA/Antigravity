# Roadmap : Suite d'Outils "Expert CPA Assistant"

Ce document détaille les outils proposés pour assister le CPA dans ses missions quotidiennes, garantissant structure, conformité et efficacité.

## 1. Module : Revue Analytique Automatisée (The "Analytic Engine")
**Objectif** : Détecter instantanément les anomalies dans les États Financiers avant même l'intervention humaine.

*   **Fonctionnalités Clés** :
    *   **Import Universel** : Importation de Balance de Vérification (Excel, CSV, PDF scanné via OCR).
    *   **Comparatif N vs N-1** : Calcul automatique des variations (montant et %) avec mise en évidence des seuils de matérialité.
    *   **Ratios de Cohérence** : Calcul automatique (Marge Brute, Liquidité, Rotation Stocks) et alerte si déviation majeure par rapport au secteur.
    *   **Flux de Trésorerie Implicite** : Reconstitution rapide du tableau de flux pour valider la cohérence de l'encaisse.

## 2. Module : Contrôle Qualité & Conformité (The "Compliance Guardian")
**Objectif** : S'assurer qu'aucun dossier ne sort sans respecter les normes (NIV/NCECF).

*   **Fonctionnalités Clés** :
    *   **Checklist Dynamique** : Génère la liste des procédures requises selon le type de mission (Audit vs Examen) et les risques identifiés.
    *   **Validateur de Signatures** : Scanne le dossier pour vérifier que tous les documents clés (Lettre de mission, Déclarations, Feuilles maîtresses) sont signés et datés.
    *   **Gestionnaire de Notes de Revue** : Interface centralisée pour laisser des points de revue à l'équipe, avec suivi de résolution bloquant la finalisation.

## 3. Module : Production de Livrables (The "Deliverable Factory")
**Objectif** : Automatiser la rédaction des rapports répétitifs.

*   **Fonctionnalités Clés** :
    *   **Générateur d'États Financiers** : Fusionne les données de la balance validée dans le gabarit Word/Excel du cabinet (Notes aux états financiers incluses).
    *   **Rapport de Mission** : Rédaction auto des paragraphes standards (Opinion, Responsabilité) selon les normes en vigueur.
    *   **Lettre de Gestion** : Bibliothèque de points d'amélioration pré-rédigés à insérer selon les faiblesses détectées.

## 4. Module : Inspecteur de Dossier "Simulé" (The "Pre-Inspection Bot")
**Objectif** : Préparer l'inspection professionnelle en simulant le regard d'un inspecteur de l'Ordre.

*   **Fonctionnalités Clés** :
    *   **Audit de Structure** : Vérifie que l'arborescence du dossier est conforme au standard 2026.
    *   **Détecteur de Vide** : Alerte si une section "Risques élevés" est vide de documentation.
    *   **Cohérence des Dates** : Vérifie que la date du rapport n'est pas antérieure à la date de fin des travaux (erreur classique).

## Priorisation Suggérée
1.  **Revue Analytique** (Gain de temps immédiat sur chaque dossier).
2.  **Contrôle Qualité** (Réduction du risque professionnel).
3.  **Production** (Confort et rapidité).
4.  **Inspection** (Outil de fond).
